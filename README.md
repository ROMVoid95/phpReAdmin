# phpReAdmin 0.1

##PHP RethinkDB Admin##
phpMyAdmin for Rethinkdb

###This project can:###
- Create/drop Rethinkdb Databases
- Create/drop Rethinkdb Tables
- Create/delete Rethinkdb Docs (rows)
- Execute Rethinkdb php Querys

###Uses:###
Bootstrap (design)
PHP-RQL (db connection)
